volcanoTest {msDiaLogue}R Documentation

Producing results for volcano plot

Description

This function produces output (fold change, and P value) for a volcano plot.

Usage

volcanoTest(x, lengthA)

Arguments

x

The function takes a parameter x, a 1d dataframe, where data from the first condition are listed first, and data from the second condition are listed second.

lengthA

The parameter "Length of A" is an integer specifying how many of the data points in x are from the first condition

Details

If the means between the two conditions are the same, t.test throws an error, which is caught and printed to the user to make sure the program keeps running

WARNING: currently, the function does not apply a further transformation. The data must be log 2 transformed in a previous step to be transformed here.

Value

The function returns a list of length 2 with the difference of means and the t-tested p-values for the protein across two conditions.
