filterOutIn {msDiaLogue}R Documentation

Filtering proteins or contaminants

Description

filterOutIn() applies a series of filtering steps to the data set

Usage

filterOutIn(dataSet, removeList, listName)

Arguments

dataSet

The 2d data set of experimental values

removeList

A Boolean specifying whether entries in the following list should be removed or selected. "TRUE" will remove the list of proteins from the data set. "FALSE will remove all proteins not in the list from the data set.

listName

A list of proteins or contaminants to select or to remove

Details

If contaminants are removed, a .csv file is created with the removed data.

Value

The function returns a filtered 2d dataframe.
