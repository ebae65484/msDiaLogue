filterOutIn {msDiaLogue}R Documentation

Filtering proteins or contaminants

Description

Apply a series of filtering steps to the data set.

Usage

filterOutIn(dataSet, removeList, listName = c(), stringSearch = "")

Arguments

dataSet

The 2d data set of experimental values.

removeList

A boolean specifying whether entries in the following list should be removed or selected.

  • TRUE: Remove the list of proteins from the data set.

  • FALSE: Remove all proteins not in the list from the data set.

listName

A list of proteins or contaminants to select or to remove.

stringSearch

A string used to match against protein names, for either inclusion or exclusion.

Details

If proteins are removed by list, a .csv file is created with the removed data.

Value

A filtered 2d dataframe.
