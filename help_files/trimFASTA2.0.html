trimFASTA2.0 {msDiaLogue}R Documentation

Trimming down a protein FASTA file to certain proteins

Description

Trimming down a FASTA file to only contain proteins present in an associated spectronaut report file.

Usage

trimFASTA2.0(
  FASTAFileName,
  reportFileName,
  outputFileName,
  selectString = "*BOVIN"
)

Arguments

FASTAFileName

A string indicating the FASTA .txt filename.

reportFileName

A string indicating the spectronaut report .csv filename.

outputFileName

A string indicating the name for the new .txt FASTA file.

selectString

A string containing a regular expression for which to search.

Details

Depending on the size of the FASTA files, this function can be slow and take multiple minutes to run. The FASTA file must be in a .txt format. No other FASTA formats will work.

Value

The function returns a FASTA file with only the specified proteins present.
