meanVariancePlot {msDiaLogue}R Documentation

Plotting a graph of mean versus variance

Description

Take a set of protein data organized by column, calculate the mean and variance of each column, and then plot those statistics.

Usage

meanVariancePlot(datMV, title = "")

Arguments

datMV

A data frame containing the data signals.

title

A string with the desired title for the mean-variance plot.

Value

An object of class plot.
