vennMain {msDiaLogue}R Documentation

Operates the venn diagram functions

Description

Builds a venn diagram, if useful, and provides universal protein list

Usage

vennMain(combos.list, showUniversal = FALSE, fileName = "test.tiff")

Arguments

combos.list

The output from sortCondition: lists of condition combinations

showUniversal

A Boolean specifying if a list of proteins that are present in every condition should also be returned.

fileName

A string, ending in .tiff specifying the name for the venn diagram

Value

The function returns nothing.
