vennMain {msDiaLogue}R Documentation

Operate the Venn diagram functions

Description

Create a Venn diagram if useful, and provide a universal protein list.

Usage

vennMain(combos.list, showUniversal = FALSE, fileName = "test.tiff")

Arguments

combos.list

The output from sortCondition, lists of condition combinations.

showUniversal

A boolean specifying whether to return a list of proteins that are present in every condition.

fileName

A string ending in .tiff, specifying the name for the Venn diagram.

Value

The unions of conditions with each proteins present.
