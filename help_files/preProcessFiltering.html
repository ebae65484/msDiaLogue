preProcessFiltering {msDiaLogue}R Documentation

Filtering of raw data signals

Description

filtering() applies a series of filtering steps to the data set

Usage

preProcessFiltering(
  dataSet,
  filterNaN = TRUE,
  filterUnique = 2,
  filterBlank = TRUE
)

Arguments

dataSet

The 2d data set of experimental values

filterNaN

A Boolean (default = TRUE) specifying whether or not proteins with NaN values should be removed from the data set.

filterUnique

An integer (default = 2) specifying whether or not proteins with less than the default number of unique peptides should be removed from the data set.

filterBlank

A Boolean (default = TRUE) specifying whether or not proteins without names should be removed from the data set.

Details

All forms of filtering are recommended for most use cases.

Value

The function returns a filtered 2d dataframe.
