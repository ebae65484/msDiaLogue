impute {msDiaLogue}R Documentation

Imputation of raw data signals

Description

impute() applies an imputation method to the data set.

Usage

impute(dataSet, imputeType = "LocalMinVal", reqPercentPresent = 51)

Arguments

dataSet

The 2d data set of experimental values

imputeType

A Boolean (default = "LocalMinVal") specifying what imputation method to use.

reqPercentPresent

An int (default = 51) specifying the require percent of values that must be present in a given protein by condition combination for values to be imputed.

Details

"LocalMinVal" replaces missing values with the lowest value from the protein by condition combination. "GlobalMinVal" replaces missing values with the lowest overall value in the data set.

Value

The function returns a list of the imputed 2d dataframe, and a shadow matrix showing which proteins by replicate were imputed.
