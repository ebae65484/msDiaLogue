analyze {msDiaLogue}R Documentation

Analyzing summarized data

Description

analyze() applies a statistical test to the data.

Usage

analyze(dataSet, conditions, testType = "t-test")

Arguments

conditions

A list of length 2, composed of integers, specifying which conditions to compare

testType

A string (default = "t-test") specifying which statistical test to use.

summaryData

The 2d data set of data

Details

Options for testType are:

"t-test" for unequal variance t-test.

"volcano" for output to plot a volcano plot.

"MA" for output to plot an MA plot.

Value

The function returns a 2d dataframe with differences of means and p-values for every protein across the two conditions. The dataframe is sorted by ascending p-value.
