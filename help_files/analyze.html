analyze {msDiaLogue}R Documentation

Analyzing summarized data

Description

Apply a statistical test to the data.

Usage

analyze(dataSet, conditions, testType = "t-test")

Arguments

dataSet

The 2d data set of data.

conditions

A list of length 2, composed of integers, specifying which conditions to compare.

testType

A string (default = "t-test") specifying which statistical test to use:

  1. "t-test": unequal variance t-test.

  2. "volcano": output to plot a volcano plot.

  3. "MA": output to plot an MA plot.

Value

A 2d dataframe with differences of means and P-values for every protein across the two conditions, sorted in ascending order by P-value.
