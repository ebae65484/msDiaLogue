tTesting {msDiaLogue}R Documentation

Producing results of t-test

Description

Produce output (difference of means, and P-value) for visual inspection or for external use.

Usage

tTesting(x, lengthA)

Arguments

x

A 1d dataframe, where data from the first condition are listed first, and data from the second condition are listed second.

lengthA

An integer specifying the number of the data points in x are from the first condition.

Value

A list of length 2 with the difference of means and the P-values from a t-test for the protein across two conditions.
